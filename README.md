# Tecnológica Antropológica

**Tecnológica Antropológica** é uma aplicação web interativa que permite aos usuários pesquisar e explorar diferentes épocas históricas, fornecendo informações sobre as tecnologias e avanços antropológicos de cada período.

## Funcionalidades

- **Busca por Época**: Através de um campo de pesquisa, o usuário pode digitar o nome de uma época ou termos relacionados e obter informações relevantes.
- **Correção Automática de Acentos**: A aplicação normaliza os acentos nas palavras pesquisadas, permitindo que os usuários realizem pesquisas sem a necessidade de incluir acentuações.
- **Informações Detalhadas**: Cada resultado inclui o nome da época, uma descrição e um link para mais informações externas.

## Tecnologias Utilizadas

- **HTML5**: Estrutura do layout.
- **CSS3**: Estilização da interface.
- **JavaScript**: Lógica de busca e normalização de acentos.
- **Normalize**: Correção automática de acentos para melhorar a experiência de busca.

## Estrutura do Projeto

- `index.html`: Página principal da aplicação, contendo o campo de busca e a área de resultados.
- `style.css`: Arquivo de estilos que define a aparência da página.
- `dados.js`: Arquivo contendo os dados das épocas históricas, incluindo tags, descrições e links.
- `app.js`: Lógica de pesquisa, normalização de acentos e exibição de resultados.

## Como Usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/tecnologica-antropologica.git
   ```
2. Abra o arquivo `index.html` em seu navegador.
3. Use o campo de pesquisa para digitar uma época ou palavra-chave (exemplo: "Paleolítico").
4. A aplicação exibirá as informações correspondentes, mesmo que você não utilize acentos.
